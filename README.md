# 🤖 AI Chat Assistant

A sleek, modern, and responsive web-based AI Chat Assistant built entirely with vanilla HTML, CSS, and JavaScript. The application connects directly to the Google Gemini API to deliver lightning-fast, intelligent, and context-aware responses right in your browser.

🌐 **Live Demo:** [sathaniya-m.github.io/chat-assistant](https://sathaniya-m.github.io/chat-assistant/)

---

## ✨ Features

* **Advanced AI Intelligence:** Powered by the new, high-performance `gemini-3.5-flash` model.
* **Modern Interface:** A clean, minimal layout featuring message bubbles, copy actions, and distinct styling for user/model roles.
* **Conversational Context:** Keeps track of ongoing conversations for coherent, natural chat flow.
* **Secure API Key Handling:** Utilizes local browser memory (`localStorage`) to query the API securely without exposing your sensitive credentials in public source code.
* **Multi-Chat Management:** Easily create new chat sessions or clear current histories with a dedicated sidebar.
* **Responsive Layout:** Works beautifully across desktop monitors and mobile screen viewports.

---

## 🚀 Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **API Integration:** Native Browser Fetch API (REST client approach)
* **Backend Framework:** Serverless / Client-side only (Hosted on GitHub Pages)
* **LLM Engine:** Google Gemini API (`gemini-3.5-flash`)
